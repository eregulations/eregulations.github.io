# Administrative interface

<a href="https://vimeo.com/147592519" target="_blank" class="btn btn-warning">See video here</a>

All the data on procedures, steps, directories of institutions, laws, forms, photos, menus, texts, etc. are registered and modified through the administrative interface.  

<img src="http://help.eregulations.org/wp-content/uploads/2013/08/4admin1.png" alt="4admin1" title="4admin1.png" class="img2" />

<h3>The administrative interface has three tabs:</h3>

<div class="row">
	<div class="col-md-9">
	<h4>Content</h4>
	<ul>
		<li>Create and edit procedures</li>
		<li>link to the database of institutions and civil servants</li>
		<li>Link to the database of requirements (administrative forms, models etc.)</li>
		<li>Link to the database of norms (laws, decrees, circulars, etc.)</li>
		<li>Library of images and documents attached to elements of the database</li>
		<li>Procedures translation</li>
	</ul>
	</div>
	<div class="col-md-3">
	<img src="http://help.eregulations.org/wp-content/uploads/2015/10/NewImage1.png" class="img2" style="padding:10px;"/>
	</div>
</div>

<div class="row">
	<div class="col-md-9">
	<h4>Public interface</h4>
	<ul>
		<li>Home page layout</li>
		<li>Create and edit menus</li>
		<li>Menus translation</li>
		<li>Contact page </li>
	</ul>
	</div>
	<div class="col-md-3">
	<img src="http://help.eregulations.org/wp-content/uploads/2015/10/public-inerface.png" alt="Public inerface" title="public inerface.png" class="img2"  style="padding:10px;"/>
	</div>
</div>

<div class="row">
	<div class="col-md-9">
	<h4>System settings</h4>
	<ul>
		<li>Users link (create user profiles and access rights) </li>
		<li>Feedback view (complaints or simplification ideas) sent by visitors of the system</li>
		<li>Country parameters (local currency, costs variables, etc.)</li>
		<li>System translation (administrative interfaces, labels, etc.)</li>
		<li>Activity log (view of users' logs/activities in the system)</li>
		<li>Maintenance of the system's media</li>
	</ul>
	</div>
	<div class="col-md-3">
		<img src="http://help.eregulations.org/wp-content/uploads/2015/10/system-settings1.png" alt="System settings" title="system settings.png" border="0"  style="padding:10px;" />
	</div>
</div>
